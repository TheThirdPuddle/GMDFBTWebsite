GMod Data Backup Tool Documentation

Introduction

This tool is created and intended for backing up your GMod \data\ folder easier, quicker, and with fewer clicks and digging through the files.

thats all there is to the introduction.

Installation

downloading from GitHub:

  1. Download the latest version of the tool from the GitHub Releases page.
  2. Extract the downloaded file.
  3. Open the folder and run the GMDFBT-x64.exe file.

downloading from the download page:

    It's the same as the GitHub method, but using this site's download page
